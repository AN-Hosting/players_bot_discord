# Discord Bot

Un bot Discord simple pour voir les joueurs connecté a votre serveur.

## Installation

1. Installez les dépendances :
    ```bash
    npm install
    ```
2. Configurez votre fichier `.env` avec votre token Discord.

## Utilisation

Lancez le bot avec :
```bash
npm start
```
